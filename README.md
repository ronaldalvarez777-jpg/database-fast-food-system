# 🗄️ Diseño lógico de base de datos – Sistema de restauración rápida

## 📌 Descripción
Este proyecto consiste en el diseño lógico de una base de datos para la gestión de un sistema de restauración rápida. Incluye el modelado entidad-relación y su estructuración lógica.

Se han aplicado principios de:
- Normalización (3FN)
- Integridad referencial
- Escalabilidad del sistema

## 🧠 Modelo de datos

El sistema se organiza en torno a la entidad **Pedido**, que conecta clientes, productos y restaurantes.
